# Assignment 4: Heap Data Structures: Implementation, Analysis, and Applications

## Overview
This repository contains the code and report for Assignment 4. It includes:

- A Heapsort implementation
- A runtime comparison of Heapsort, Quicksort, and Merge Sort
- A priority queue implemented with a binary max-heap
- A scheduler simulation using the priority queue
- A written report analyzing the algorithms and design choices

## Files
- `heapsort.py` - Implements Heapsort using a max-heap
- `sorting_comparison.py` - Compares Heapsort, Quicksort, and Merge Sort
- `priority_queue.py` - Implements a max-heap priority queue with a Task class
- `scheduler_simulation.py` - Demonstrates a scheduling simulation using the priority queue
- `report.md` - Written analysis and discussion
